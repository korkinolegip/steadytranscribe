# SteadyTranscribe — локальная транскрипция файлов для Windows

## Проблема
Олегу (и коллегам в SteadyControl) нужно превращать аудио/видео-записи (встречи, звонки) в текст на Windows-компьютерах. Сейчас это умеет только FluidVoice на его Mac (функция File Transcription). Облачные сервисы не подходят: записи конфиденциальны (внутренние встречи компании), а облачные AI-API из сети ненадёжны (блокировки по IP).

## Решение
Отдельное Windows-приложение (Windows 10/11), воспроизводящее функцию File Transcription из FluidVoice один в один:
1. Перетащил аудио/видео-файл в окно (или выбрал кнопкой)
2. Локальное распознавание (модель Whisper на компьютере, аудио никуда не уходит)
3. Готовый текст: скопировать/экспортировать
4. История всех транскрипций с датой и временем
5. Настройки: модель (размер/качество), язык, папка экспорта

## Почему сейчас
FluidVoice существует только для macOS (Swift/CoreML — непереносимо). Локальные модели распознавания (faster-whisper) достигли качества, достаточного для русской речи, и работают на обычном офисном CPU.

## Аудитория
- Основная: Олег и сотрудники SteadyControl на Windows 10/11 (не разработчики — нужен обычный установщик .exe, всё ставится само)
- Вторичная: любые русскоязычные пользователи с потребностью в приватной транскрипции

## Архитектура
```
┌──────────────────────────────────────────────┐
│ UI (PySide6/Qt): drag&drop, прогресс,        │
│ история, настройки — тёмная/светлая тема     │
├──────────────────────────────────────────────┤
│ Ядро: конвертация (ffmpeg, встроен)          │
│ → 16 kHz mono WAV → faster-whisper           │
│ (CTranslate2, int8, CPU; CUDA если есть GPU) │
├──────────────────────────────────────────────┤
│ Хранилище: SQLite (история) + JSON-настройки │
│ %APPDATA%/SteadyTranscribe/                  │
└──────────────────────────────────────────────┘
```
Стек: Python 3.12 + PySide6 + faster-whisper + ffmpeg (вшит) → PyInstaller → Inno Setup (единый установщик .exe). Сборка — GitHub Actions (windows-latest), локально Windows-машины у нас нет.

Обоснование: faster-whisper — лучшее локальное распознавание русского на Windows-CPU; PySide6 — нативный вид, надёжный drag&drop; Inno Setup — стандарт установщиков Windows.

## Монетизация
Нет — внутренний инструмент.

## Конкуренты
| Конкурент | Что делает | Чего не хватает |
|---|---|---|
| FluidVoice | эталон функции | только macOS |
| Whisper Desktop / Buzz | локальная транскрипция на Windows | англ. интерфейс, нет нужного UX истории, лишние функции |
| Облачные (tl;dv, Fireflies) | транскрипция встреч | не локально, блокировки, конфиденциальность |

## План запуска
- **MVP (v1)**: drag&drop + выбор файла → распознавание → текст; история (SQLite); копирование; настройки (модель, язык, устройство CPU/GPU); прогресс; установщик
- **v2**: экспорт с таймкодами (SRT/VTT), очередь из нескольких файлов, автоопределение колонок/спикеров — только если попросит Олег

## Риски
| Риск | Вероятность | Митигация |
|---|---|---|
| Скорость на слабом офисном CPU | средняя | модель small/int8 по умолчанию, выбор модели в настройках |
| Сборка установщика без Windows-машины | средняя | GitHub Actions windows-latest; тест — на машине Олега |
| Большой размер установщика (модель) | высокая | модели скачиваются при первом запуске (как в FluidVoice), установщик лёгкий |
| Антивирусы ругаются на PyInstaller | средняя | подписи нет; инструкция + при проблемах перейти на Inno-only раскладку |
